1/ On a créer un dossier sous le nom "learn_git_again"

2/ On a acceder au dossier crée 

3/ On a créer du fichier appelé trois.txt

4/ On a initialiser un dossier git vide avec la commande ---->  git init

5/ Ajout du fichier trois.txt a staging area avec la commande ----> git add trois.txt

6/ Commit du fichier avec le message (ajout de trois.txt) on utilisant la commande ----> git commit -m "Ajout de trois.txt"

7/ Verification avec la commande ----> git log

8/ On a créer un fichier nommé "quatre.txt"

9/ Ajout du fichier trois.txt a staging area avec la commande ----> git add quatre.txt

10/ Commit du fichier avec le message (ajout de quatre.txt) on utilisant la commande ----> git commit -m "Ajout de quatre.txt"

11/ Suppression du fichier trois.txt avec la commande ----> git rm trois.txt

13/ Commit de l'action avec le message (suppression de trois.txt) on utilisant la commande ----> git commit -m "Suppression de trois.txt"

14/ Verification du commit avec la commande ----> git log
